.. _support:

Support
=======

As an open-source project, we try to keep all of our support requests on GitHub. If you

#. have a question that is not answered by this documentation,
#. discover a bug in the software, or
#. would like to request a feature in the software

then please `create an issue <https://github.com/Eco-Libre/genesis-booth/issues/new/choose>`_ on our GitHub.

If you don't want to register for an account on GitHub or you'd like to remain anonymous, then you can `email us <https://www.eco-libre.org/contact/>`_. We'll put the issue on GitHub for you.
